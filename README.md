# Winraycast
This is a refactoring of 1990s ray-cast 3d engine written in C++ running on Windows.
It is not based on OpenGL or DirectX or other 3D library. 
Green field approach used to demostrate the ray-cast algorithm and related to an article published on Computer Programming magazine by Infomedia.
